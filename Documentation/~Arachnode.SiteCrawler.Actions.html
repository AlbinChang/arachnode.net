Class Description
Renderer The Renderer plugin is used to process DecodedHtml form rendered CrawlRequests. The primary usage of rendering a WebPage is to obtain HyperLinks to Discoveries that are not present in the DecodedHtml when downloaded with the WebClient, or when viewing the source from a browser.
